# La-avenida-de-los-Yankees
⚾ La Avenida de los Yankees - Fan Page Proyecto de práctica: Integración avanzada de HTML5 y CSS3.  🚀 Desafíos Técnicos Superados: Identidad Visual con CSS Puro: Implementé el icónico diseño de "pinstripes" (rayas verticales) de los Yankees usando linear-gradient en el fondo, evitando el uso de imágenes pesadas.  Layout Complejo con CSS Grid: Utilicé una estructura de rejilla para separar la noticia principal de los elementos laterales (infografías y estadísticas).  Diseño Responsivo (Mobile First): Implementé un contenedor de video (.iframe-responsive) que mantiene la relación de aspecto 16:9 en cualquier tamaño de pantalla.  Tipografía y Jerarquía: Uso de fuentes externas (Oswald) y la función clamp() para que el logo se adapte fluidamente al tamaño del navegador.
